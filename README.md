## Featured Projects

### OrderSphere · Order and Delivery Management System

OrderSphere is a comprehensive management platform built for a meat distribution business. It governs the full commercial lifecycle of an order, from initial customer capture through inventory allocation, warehouse picking, dispatch, delivery, and executive reporting.

**Architecture:** The platform follows a layered architecture. An ASP.NET Core Web API  communicates with a SQL Server database through Entity Framework Core. The primary interface is an Angular application that presents operational dashboards and management screens, with Chart.js driving the reporting visualisations. A dedicated Ionic and Capacitor application serves the warehouse floor, giving staff a tablet interface for picking and stock tasks. Authentication and authorisation are handled through JWT, with access governed by user role so that each user type sees only what is relevant to them.

**Reporting:** The reporting layer reads from purpose built SQL views that aggregate the key operational metrics, including daily order volume, weekly revenue, open deliveries, and stock on hand. These views feed an executive dashboard composed of Angular Material components and Chart.js visualisations, giving management a live operational picture at a glance.

**My role:** I served as the lead developer and acted as the project manager for the team of 5 that built the project. 

The source code is private, as the project is tied to both a team and a live business. I am glad to present the architecture in detail and walk through the code during an interview.
